# Hardwave KickForge

Hardstyle & hardcore kick synthesizer — VST3/CLAP plugin.

**Status: Pre-Alpha**

## Features (planned)
- Layered kick synthesis (click + body + sub)
- Pitch envelope with exponential sweep
- Drive/distortion per layer
- Tail tuning for hardstyle kicks
- Built-in from Hardwave Suite

## Build
```bash
cargo xtask bundle hardwave-kickforge --release
```
